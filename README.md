## Enum Type Typescript

```
// this is a mixed enum type
// two others enum type is number and string enum type
enum mixedType {
  name = "shihab",
  age = 1,
}
console.log(mixedType["age"]);
console.log(mixedType.age);
console.log(mixedType);

```
